# AI Image Generation Prompts for Recipe Placeholders

## General Template

Use this template with AI image generation models (DALL-E, Midjourney, Stable Diffusion, etc.):

```
Create a high-quality food photograph of [DISH_TYPE] in a cartoonish, stylized art style. The image should be 800x600 pixels, well-lit, appetizing, and centered in the frame. Use vibrant colors and clean composition. The style should be slightly animated/illustrated but still realistic enough to represent actual food. No text, watermarks, or people in the image.
```

## Tips for Best Results

1. **Use these prompts with popular AI image generators:**
   - DALL-E 3 (OpenAI)
   - Midjourney
   - Stable Diffusion
   - Adobe Firefly

2. **Adjust the style keywords if needed:**
   - For more realistic: "photorealistic food photography with slight stylization"
   - For more cartoon: "animated food illustration, Disney-style"
   - For artistic: "food art illustration, vibrant and stylized"

3. **Quality settings:**
   - Always specify 800x600 or similar aspect ratio
   - Request high resolution/quality
   - Mention good lighting and composition

4. **Consistency tips:**
   - Use the same AI model for all images to maintain style consistency
   - Keep the same lighting and composition keywords across all prompts
   - Generate a few variations and pick the best ones

5. **File naming:**
   - Save generated images with the exact filenames specified in the system
   - Convert to .jpg format if the AI generates PNG files
   - Ensure 800x600 pixel dimensions for consistency
